# Animação com Canvas

Projeto de aplicação simples desenvolvida com **HTML, CSS e JavaScript**, consiste em uma animação com círculos, que trocam de cor ao clique do usuário. <br/>
Atividade do módulo **Desenvolvimento em Javascript**, do curso **Ciência da Computação**.
